# Living with a Bear - Official Band Website

A retro 90s/2000s Windows 97-style band website for Living with a Bear.

## Features

- 🎸 **Retro Windows 97 Aesthetic** - Classic taskbar, windows UI, and sound effects
- 🎵 **Music Player** - Interactive music player with playback controls
- 📷 **Media Gallery** - Links to Instagram and YouTube
- 🎤 **Show Dates** - Upcoming tour dates and venues
- 📧 **Contact Form** - Get in touch with the band
- 🔊 **Click Sounds** - Classic Windows 97 sound effects
- 📱 **Social Media Integration** - Direct links to all social platforms

## Sections

1. **Home** - Welcome message and band bio
2. **About** - Band story and sound description
3. **Photos & Videos** - Links to social media content
4. **Music** - Music player and upcoming tracks
5. **Show Dates** - Upcoming tour dates
6. **Contact** - Contact form and direct contact info

## Technologies Used

- HTML5
- CSS3 (Windows 97 styling)
- Vanilla JavaScript
- Web Audio API (for sound effects)

## How to Use

1. Deploy this website to GitHub Pages
2. Point your Cloudflare domain to GitHub Pages
3. Add your real content (music links, show dates, photos)

## Customization

### Add Music
Replace the placeholder tracks in the Music section with actual audio links or embedded players.

### Update Show Dates
Edit the show dates section with real venue information and ticket links.

### Add Photos
Add image links from Instagram or create an image gallery.

### Change Social Media Links
Update all social media URLs with your actual band links.

## GitHub Pages Setup

1. Push this to your GitHub repo
2. Go to Settings > Pages
3. Select 'Deploy from a branch'
4. Choose 'main' branch
5. Your site will be live at `https://your-username.github.io/livingwithabear`

## Custom Domain Setup

1. In Cloudflare, add DNS records:
   - Type: CNAME
   - Name: www
   - Content: your-username.github.io
   
2. In GitHub Pages settings, add your custom domain
